손흥민은 선발 출격했고, 해리 케인과 브라이언 힐과 호흡을 맞추면서 공격 작업에 열중했다.

토트넘은 전체적으로 답답했고, 뜻대로 풀리지 않았다. 손흥민은 전반 18분 상대의 공격을 차단한 후 답답한 표정을 지으면서 마스크를 벗어 던졌다.
